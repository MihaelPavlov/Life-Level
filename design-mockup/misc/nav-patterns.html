PATTERN 4
World vs Me — 5 Smart Tabs
Groups destinations semantically. "World" tab = Map + Guild. "Me" tab = Profile + Stats + Titles. Each tab has internal sub-navigation.

✓ 5 tabs, none feel random or forced
✓ Scales well as features grow
✗ "World" and "Me" may need onboarding
✗ Sub-nav adds slight complexity inside tabs
