⚡ XP & Stat Calculation Engine

All logic lives in ActivityService.CalculateGains() — hardcoded constants, no DB-driven multipliers.

// Step 1 — Base XP from duration
baseXp = durationMinutes × 3.0

// Step 2 — Distance bonus (activity-specific, per km)
if (distanceKm > 0 && hasDistanceBonus)
    baseXp += distanceKm × distanceBonusPerKm

// Step 3 — Calorie bonus
if (calories > 0)
    baseXp += calories / 10.0

// Step 4 — Apply activity type multiplier
finalXp = Round(baseXp × typeMultiplier)

Reprocess-Stuck Logic

Endpoint: POST /integrations/health/reprocess-stuck

Scenario A — Torn write
Sentinel exists (WasImported=false) but Activity row does exist for ExternalId
→ Repair: set WasImported=true in-place (no XP re-awarded)
Scenario B — True failure
Sentinel exists (WasImported=false) and no Activity row exists
→ Report error — needs fresh sync from provider
Race condition protection
Unique constraint on (CharacterId, Provider, ExternalId) blocks concurrent webhooks from creating duplicate sentinels
