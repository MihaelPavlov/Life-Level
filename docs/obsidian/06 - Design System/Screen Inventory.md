## How to use these

- Mockups are **design references**, not production HTML.
- Flutter implementation diverges from some mockup details — mockups show *intent*, Flutter screens show *current reality*.
- When adding a new feature, check for an existing mockup first — `fitness-mode-v6.html` has been through 6 iterations for a reason.
