Entry zone

The region's starting zone. Tick Start zone above — only one Entry zone per region. No type-specific config.

Standard zone

A named story zone on the critical path. Players traverse edges to reach it. XP reward above = zone-completion XP.

Chest zone

Side reward. Name + Emoji above = chest display. XP reward above = zone XP on arrival; Chest reward XP below = bounty on open.

Dungeon zone

Multi-floor run. Each floor requires a specific activity (e.g. Running 2 km) to clear. Bonus XP awarded on full clear, in addition to per-floor rewards.

Floors
Save the zone first — then floors can be added.

Boss zone

Name + Emoji above = boss display. XP reward above = victory bounty. HP is auto-computed from region chapter + zone tier.

Crossroads zone

Branching point. Create two downstream zones and set their Branch of field to this Crossroads. Players will pick one branch.
